Hierarchical Vision-Language Reasoning for Multimodal Multiple-Choice Question Answering

저자 · Ao Zhou, Zebo Gu, Tenghao Sun, Jiawen Chen, Mingsheng Tu, Zifeng Cheng, Yafeng Yin, Zhiwei Jiang, Qing Gu 출처 · Proceedings of the 33rd ACM International Conference on Multimedia (MM '25), Dublin, Ireland 연도 · 2025

핵심 요약

이 논문은 복잡한 레이아웃과 방대한 분량을 가진 일본어 PDF 문서에서, 10개의 선택지 중 정답을 고르는 ten-choice 질의응답(multiple-choice QA) 과제를 푸는 프레임워크를 제안한다. 핵심 아이디어는 (1) ColQwen2(ColPali 계열)를 이용한 vision-language 검색으로 전체 PDF에서 질의와 가장 관련 있는 상위 3개 페이지 이미지를 찾고 QwenVL2.5로 재필터링한 뒤, (2) 대규모 언어모델과 정교한 프롬프트로 원 질문을 여러 sub-question으로 분해(sub-question decomposition)하고 그 답을 검증하는 계층적(hierarchical) 추론을 수행하며, (3) bounding box 기반 crop 모듈로 검색 결과를 보정하는 것이다. 프롬프트는 언어 편향을 줄이기 위해 1단계는 영어·일본어 이중언어로, 2단계는 일본어 전용으로 설계되고, think/answer 필드를 가진 구조화된 JSON으로 추론 과정을 함께 출력한다. 저자들은 task-specific fine-tuning 없이 여러 최신 MLLM을 zero-shot으로 Public Score 지표에서 비교하였고, Qwen2.5-VL-72B·Kimi-VL-A3B-Thinking·GLM-4.1-9B-Thinking을 결합한 fusion이 0.59로 최고 성능을 기록했다. 실패 사례 분석에서는 retrieval precision이 주요 병목임을 확인하였다.

해결하려는 문제

논문 주장

긴 PDF 문서에 대한 ten-choice 질의응답

MLLM(Multimodal Large Language Model)은 시각·텍스트 정보를 통합해 VQA(Visual Question Answering)에서 뛰어난 능력을 보였으나, 10개 선택지 중 정답을 고르는 어려운 ten-choice 평가 방식 하에서 수십~수백 페이지에 이르는 방대한 분량과 표·차트·다이어그램 등 비선형적이고 복잡한 레이아웃을 가진 PDF 문서를 처리할 때 여전히 유의미한 한계를 보인다. 이러한 문서 처리 능력은 상업적 배포의 핵심 장벽으로 지적된다.

PDF p. 1

비영어권(일본어) 언어 격차

현재 주류 모델들은 영어 학습 데이터에 강하게 편중되어 있어 일본어를 비롯한 비영어권 언어 시나리오에서 최적 이하의 성능을 보인다. 특히 일본어 문서 수준 VQA는 JDocQA가 유일하게 공개된 벤치마크일 정도로 연구가 부족한 영역이다.

PDF p. 1, 2

기존 방식의 한계

논문 주장

OCR·순수 텍스트 모델의 파싱 한계

전통적인 OCR(Optical Character Recognition) 기법과 순수 텍스트 기반 언어모델은 표·차트 등 구조화된 문서 콘텐츠를 효과적으로 파싱하는 데 어려움을 겪으며, 종합적인 문서 이해는 요소 간 의미적 상관관계를 확립해야 하는 경우가 많다.

PDF p. 2

문서 특화·범용 모델의 잔존 한계

mPLUG-DocOwl, Docopilot 같은 문서 이해 특화 VLM과 GPT-4, GLM4.1, Qwen-VL, Kimi-VL 등 범용 MLLM이 문서 이미지 처리에서 점차 향상된 적응력을 보여왔음에도, 어려운 ten-choice 평가 방식 하에서 복잡한 레이아웃과 긴 콘텐츠의 PDF를 처리할 때 여전히 유의미한 한계를 남기고 있으며, 일본어 문서 수준 VQA는 특히 연구가 부족하다.

PDF p. 1, 2

핵심 contribution

논문 주장

ColQwen 최적화 검색 + 계층적 추론 프레임워크

ColQwen에 최적화된 검색 기법과 멀티모달 계층적(hierarchical) 추론 메커니즘을 결합한 새로운 일본어 PDF 문서 이해 프레임워크를 제안한다. ColQwen을 사용해 전체 PDF 문서에서 질의와 가장 의미적으로 관련성이 높은 상위 3개 페이지를 식별한다.

PDF p. 1, 2

sub-question decomposition 기반 semantic verification

대규모 언어모델과 정교하게 설계된 프롬프트로 원 질문을 일련의 sub-question으로 분해하고, semantic drift를 완화하는 semantic verification 메커니즘을 도입하여 검색 결과의 정확성과 견고성을 향상시킨다.

PDF p. 1, 2

구조화된 추론 출력과 zero-shot 평가

완전한 추론 과정(reasoning trace)을 보존하는 구조화된 출력 형식을 prompt engineering으로 생성하고, 이 프레임워크를 이용해 다수의 최신 MLLM에 대해 task-specific fine-tuning 없이 zero-shot 평가를 수행하여 일반화 능력을 Public Score 지표로 공정하게 비교한다.

PDF p. 2

제안 방식의 전체 구조

논문 주장

3개 핵심 단계 + 보조 모듈

프레임워크는 (1) ColQwen2 기반 vision-language 검색으로 질의 관련 PDF 페이지 이미지를 식별한 뒤 QwenVL2.5로 재필터링하는 검색 단계, (2) 이미지·질문·참고자료를 바탕으로 원 질문을 하위 질문들로 분해하고 답을 검증하는 2단계 sub-question decomposition 및 prompt design 단계, (3) QwenVL2.5로 정답 관련 영역의 bounding box를 추정·crop하여 검색 결과를 보정하는 보조 모듈로 구성된다(Figure 1).

PDF p. 3, 4, 5

구조화된 JSON 출력

최종적으로 think(추론 과정)와 answer(최종 답) 필드를 포함하는 구조화된 JSON 형태로 추론 과정과 최종 답을 함께 출력하여 추론 추적성과 시스템 통합을 향상시킨다.

PDF p. 5

단계별 동작 과정

논문 주장

1단계: 페이지 이미지화와 top-3 검색

PDF 문서를 개별 페이지로 분할하고 각 페이지를 PNG 이미지로 변환한 뒤, 주어진 질의에 대해 ColQwen2 모델이 유사도 점수 기준 상위 3개 페이지 이미지를 검색한다.

PDF p. 3

2단계: QwenVL2.5 재필터링

QwenVL2.5 모델이 검색된 상위 3개 이미지를 다시 검토해 관련 없는 페이지를 필터링하며, 이 과정에서 1~3개의 관련 이미지가 유지되고 최소 1개의 이미지는 항상 보존된다.

PDF p. 3

3단계: 1단계 sub-question 생성(영·일 이중언어)

1단계 프롬프트(영어+일본어 버전)로 이미지와 질문을 분석하여 여러 sub-question과 각각의 답, 그리고 참고자료(reference)를 생성한다. sub-question은 관련 데이터 위치 파악, 수치 추출, 추세 식별, 비교 분석 등 다단계 추론 유형을 포괄한다.

PDF p. 3, 4

4단계: 2단계 통합 추론(일본어 전용)

2단계 프롬프트(일본어 전용)로 원본 이미지, 질문, 1단계에서 생성된 참고자료 및 sub-question-답변 쌍을 결합하여 추가 sub-question과 답을 생성하고, 최종적으로 원 질문에 대한 최종 답을 도출한다.

PDF p. 4, 5

5단계(보조): bounding box crop 보정

QwenVL2.5로 정답에 필요한 최소 bounding rectangle 좌표 [x1,y1,x2,y2]∈[0,1]^4와 신뢰도 점수 α를 추정해 이미지를 crop하고, ColQwen으로 crop 이미지와 기존 top-3 이미지의 유사도를 비교해 crop 이미지가 더 높으면 검색 집합의 해당 이미지를 crop 이미지로 교체한다.

PDF p. 5

6단계: voting 기반 최종 결정

추론 시 질문당 3회의 독립적인 reasoning trial을 수행하며 매번 선택지 순서를 섞고, 최소 2표를 얻은 답을 최종 답으로 선택한다. 각 답이 1표씩인 동점의 경우 모든 후보 선택지를 유지한 채 2차 질의를 수행하고 1회 추가 추론으로 최종 결정을 내린다.

PDF p. 5

핵심 알고리즘 또는 인덱스 구조

논문 주장

ColBERT 스타일 late interaction 검색 (Eq.1)

ColPali는 문서 페이지 스크린샷을 patch로 나눠 ViT로 인코딩하고, patch 임베딩을 언어모델의 soft token으로 취급하여 128차원 공간으로 투영한 multi-vector 표현을 만든다. 검색은 ColBERT 스타일 late interaction으로 계산되며, LI(q,d) = Σ_{i=1}^{N_q} max_{j∈[1,N_d]} ⟨E_q^(i), E_d^(j)⟩ 로 질의 토큰마다 문서 토큰과의 최대 내적을 합산한다. 학습은 in-batch negative sampling과 softplus contrastive loss(Eq.2)를 사용한다.

PDF p. 3

ColQwen2의 개선점

ColQwen2는 ColPali 위에 종횡비를 보존하는 동적 이미지 해상도 지원, LoRA·8-bit optimizer를 이용한 효율적 학습을 더해 다양한 레이아웃에 대한 견고성을 높이면서 ColBERT 스타일 multi-vector late interaction을 유지하여 질의-문서 patch 간 fine-grained 매칭을 가능하게 한다.

PDF p. 3

sub-question decomposition 로직

이미지(차트·표·다이어그램), 질문, 참고자료를 입력받아 이미지 인식과 텍스트 추출을 결합해 구조화된 데이터를 얻고, 질문을 데이터 위치 파악·수치 추출·추세 식별·비교 분석 등의 논리적 단계로 의미적으로 분해한다. 각 sub-question에 일관된 답을 부여하지만 모듈의 최종 출력은 원 질문에 대한 단일 결론적 답이다. Figure 2의 예에서는 환경 문제 인식 조사 차트에 대해 '2014년 동의 비율은?'(81.3%) 등의 sub-question이 생성된다.

PDF p. 3, 4

실험 구성

논문 주장

데이터셋

평가는 JDocQA corpus를 기반으로 구축된 도메인 특화 일본어 PDF 문서 이해 벤치마크에서 수행되며, 10개 선택지 중 정답을 고르는 ten-choice 형식이다. 본문에서는 challenge dataset / challenge public dataset로 지칭된다. 다만 정식 명칭, 총 질문 수, 문서 수 등 구체적 규모 통계는 본문에 명시되지 않았다.

PDF p. 2, 5

비교 대상 모델과 지표

GPT-4o, Gemini-1.5, Claude-3.5, Llama-3.2-V-90B-Instruct, InternVL3-8B/9B/78B, Qwen2.5-VL-7B, Kimi-VL-A3B, GLM-4.1-9B, Qwen2.5-VL-72B(-Instruct) 등 단일 MLLM과 이들의 fusion 조합을 Public Score 지표로 비교한다. 모든 평가는 task-specific fine-tuning 없이 zero-shot으로 수행되며, 질문당 3회 reasoning trial과 선택지 셔플링을 포함한 voting mechanism으로 결과를 얻는다.

주요 결과와 의미

논문 주장

Fusion이 최고 성능

Qwen2.5-VL-72B, Kimi-VL-A3B-Thinking, GLM-4.1-9B-Thinking을 결합한 fusion이 Public Score 0.59로 전체 최고 성능을 달성했다. 단일 모델 중에서는 Qwen2.5-VL-72B와 Qwen2.5-VL-72B-Instruct가 각각 0.57로 최고였다. 두 값이 같아 instruction tuning이 본 과제에서 유의미한 향상을 주지 못함을 시사한다(Table 1).

PDF p. 5, 6

규모보다 아키텍처·데이터 품질

Qwen2.5-VL-7B(0.56)와 Kimi-VL-A3B(0.56) 같은 소규모 모델이 InternVL3-78B(0.54)나 Llama-3.2-V-90B-Instruct(0.53) 같은 더 큰 모델보다 높은 점수를 기록하여, 파라미터 규모보다 모델 아키텍처와 학습 데이터 품질이 더 중요함을 시사한다. 한편 InternVL3 계열(8B+9B+78B) 앙상블은 계열 내 최고 단일 성능 0.54를 0.56으로 끌어올렸으나, Qwen2.5-VL-72B+Kimi(0.58)에 Qwen2.5-VL-72B-Instruct를 더해도 0.58로 유지되는 한계 효과(threshold effect)가 관찰됐다.

PDF p. 5, 6

구성요소 ablation

GLM-4.1 기준 ColQwen 검색+Qwen2.5-VL 필터링만 사용한 baseline은 0.52, sub-question decomposition 추가 시 0.53, 1단계 다국어(영·일) 프롬프트 추가 시 0.54로 향상됐다. 2단계 프롬프트 개선과 추가 모듈은 이 peak를 유지하여, 성능 향상이 주로 retrieval filtering·decomposition·다국어 프롬프트 설계의 시너지에서 발생함을 시사한다(Figure 5).

PDF p. 6

검색 정밀도가 병목

10개 대표 오답 사례에서 사람이 지정한 핵심 영역(key region) 수를 1개에서 3개로 늘리자 시스템 정확도가 50%에서 80~90%로 크게 향상됐다. 이는 기존 모델이 관련 시각 정보가 주어지면 충분히 추론할 수 있으나, 현재 자동 검색 시스템이 핵심 영역을 스스로 찾아내는 데 결함이 있음을 드러낸다(Figure 6).

논문 주장

요약과 시사점

본 연구는 새로 구축한 벤치마크를 기반으로 멀티모달 정보 처리와 계층적 추론을 통합한 일본어 PDF 문서 이해 프레임워크를 설계했다. ColQwen을 이용한 정밀한 image-text 검색과 sub-question 기반 semantic verification으로 문서 레이아웃 복잡성과 장거리 의존성을 다루며, 복잡한 질의를 분해하고 구조화된 추론 과정을 생성한다. 실험은 프레임워크의 효과성과 해석 가능성을 보였고, 향후 개선 방향으로 검색 정밀도가 지목된다.

전체 동작 흐름

PDF p. 3, 4, 5

PDF를 페이지 이미지로 변환한 뒤 ColQwen2 top-3 검색 → QwenVL2.5 재필터링 → 1단계(영·일) sub-question 생성 → 2단계(일본어) 통합 추론 → voting으로 최종 답을 산출하며, bounding box crop 보정 모듈이 검색 집합을 개선한다(Figure 1 기반).

논문의 강점

  • 분석자 해석 검색(retrieval)과 계층적 추론(decomposition)을 명확히 분리·결합하고 bounding box crop 보정까지 더해, ablation에서 각 구성요소의 기여를 단계적으로 검증한 점이 방법의 설득력을 높인다.
    PDF p. 6
  • 분석자 해석 영어·일본어 이중언어 프롬프트로 언어 편향을 다루는 접근은 비영어권 문서 VQA라는 저연구 영역에 대한 실용적 대응으로, ablation에서 0.53→0.54 향상으로 효과가 뒷받침된다.
    PDF p. 5, 6
  • 논문 주장 task-specific fine-tuning 없이 여러 최신 MLLM을 동일 프레임워크로 zero-shot 평가하여, 모델 간 일반화 능력을 상대적으로 공정하게 비교할 수 있게 했다.
    PDF p. 2, 5
  • 분석자 해석 think/answer JSON 출력과 sub-question 추적으로 추론 과정의 투명성·해석 가능성을 확보한 점은 실무 통합과 답변 근거 추적에 유리하다.
    PDF p. 5

한계

  • 저자 명시 한계 저자들은 competition 학습 데이터가 풍부하고 고품질임에도 의도적으로 instruction tuning을 피하고 zero-shot을 채택했다. 실제 instruction tuning을 시도했으나 성능이 zero-shot보다 낮았으며, 원인으로 (1) 후보 답안 수가 많아 과제 난이도가 높은 점, (2) fine-tuning 데이터가 제한적인 점, (3) 전체 PDF 기반 질문과 학습용 PNG 이미지 간 불일치를 제시했다.
    PDF p. 6
  • 저자 명시 한계 10개 실패 사례 분석 결과 retrieval precision이 주요 성능 병목으로 확인됐다. 핵심 영역을 1→3개로 늘리면 정확도가 50%→80~90%로 향상된다는 점은, 모델의 추론 능력은 충분하나 자동 검색 시스템이 핵심 영역을 스스로 찾는 데 결함이 있음을 드러낸다.
    PDF p. 6
  • 분석자 해석 평가 데이터셋의 정식 명칭·질문 수·문서 수 등 규모 통계와 Public Score의 정확한 계산 방식이 본문에 제시되지 않아, 결과의 절대적 수준과 재현성을 판단하기 어렵다.
    PDF p. 2, 5
  • 분석자 해석 본문 서술의 모델명(예: Kimi-VL-A3B-Thinking)과 Table 1 헤더(Kimi-VL-A3B)가 일치하지 않고, Figure 5 baseline(0.52)과 Table 1의 GLM-4.1-9B 단일 점수(0.55)가 달라, 실험 조건 간 정합성이 본문만으로는 명확하지 않다.
    PDF p. 5, 6

핵심 용어

ten-choice question / multiple-choice QA
10개의 후보 선택지 중에서 정답 하나를 고르는 질의응답 평가 방식으로, 후보 수가 많아 과제 난이도가 높다. PDF p. 1, 2
ColPali / ColQwen2
문서 페이지 스크린샷을 patch로 나눠 ViT로 인코딩하고 이를 언어모델의 soft token으로 다루는 vision-language 검색 모델. ColQwen2는 동적 해상도, LoRA·8-bit optimizer 등으로 개선된 변형이다. PDF p. 3
late interaction (ColBERT-style)
질의와 문서를 각각 다중 벡터로 표현한 뒤, 질의 토큰마다 문서 토큰과의 최대 내적을 구해 합산하는 fine-grained 매칭 방식(Eq.1). PDF p. 3
sub-question decomposition
원 질문을 데이터 위치 파악·수치 추출·추세 식별·비교 분석 등의 논리적 하위 질문으로 분해하고 각각에 답한 뒤 단일 결론 답을 도출하는 계층적 추론 전략. PDF p. 3, 4
semantic verification / semantic drift
sub-question과 참고자료를 지식 앵커로 삼아 검색·추론 과정에서 의미가 원 질문에서 벗어나는 현상(semantic drift)을 완화하고 검색 결과의 정확성·견고성을 높이는 메커니즘. PDF p. 1, 2, 4
Public Score
challenge (public) dataset에서 모델을 비교하는 데 사용된 성능 지표. 구체적 계산 방식은 본문에 명시되지 않음(확인되지 않음). PDF p. 5
voting mechanism
질문당 3회 독립 추론을 선택지 순서를 섞어 수행하고 최소 2표를 얻은 답을 채택하며, 동점 시 2차 질의와 추가 추론으로 최종 결정하는 방식. PDF p. 5
bounding box 사전 고정 모듈
QwenVL2.5로 정답에 필요한 최소 경계 사각형 좌표와 신뢰도 α를 산출해 이미지를 crop하고, ColQwen 유사도가 더 높으면 검색 이미지를 교체해 정합성을 높이는 보조 모듈. PDF p. 5

확인이 필요한 내용

  • 평가 데이터셋('challenge dataset' / 'challenge public dataset')의 정식 명칭, 총 질문 수, 문서(페이지) 수 등 규모 통계는 본문에 명시되지 않음.
  • 본문(Section 4.2)의 'Kimi-VL-A3B-Thinking', 'GLM-4.1-9B-Thinking', 'Qwen2.5-VL-7B-Instruct'와 Table 1 헤더('Kimi-VL-A3B', 'GLM-4.1-9B', 'Qwen2.5-VL-7B')가 동일 변형인지 본문만으로 확인 불가.
  • Figure 5 ablation의 7개 지점 중 본문에는 0.52/0.53/0.54만 명시되어 EN Prompt·2단계 프롬프트·추가 모듈 등 나머지 지점의 정확한 수치는 확정 불가.
  • 'Public Score' 지표의 구체적 계산 방식·정의는 본문에 없음.
  • Figure 6에서 모델별(InternVL3-9B, Qwen2.5-VL-7B, Kimi-VL-A3B, GLM-4.1-9B) base/top1~3 지점별 정확 수치는 그래프에만 있고 본문은 '50%→80~90%' 범위만 서술.
  • Figure 5 캡션의 GLM-4.1 baseline(0.52)과 Table 1의 GLM-4.1-9B 단일 점수(0.55)가 달라 두 실험 간 관계가 본문에서 명확히 설명되지 않음.
